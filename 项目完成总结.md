## 🔧 配置和自定义

### AI助手配置

#### 选项1: Claude API（推荐功能最强）
```javascript
// src/main/services/AIService.ts
const CLAUDE_API_KEY = process.env.CLAUDE_API_KEY;

async function executeCommand(command) {
  const response = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'x-api-key': CLAUDE_API_KEY,
      'content-type': 'application/json'
    },
    body: JSON.stringify({
      model: 'claude-opus-4-1',
      max_tokens: 1000,
      messages: [{
        role: 'user',
        content: `用户命令：${command}`
      }]
    })
  });
  return response.json();
}
```

#### 选项2: 本地Ollama（离线，免费）
```bash
# 1. 下载和安装 Ollama
# https://ollama.ai

# 2. 下载模型（可选，默认会自动下载）
ollama pull llama2-chinese

# 3. 启动Ollama服务
ollama serve

# 4. 在代码中调用
fetch('http://localhost:11434/api/generate', {
  method: 'POST',
  body: JSON.stringify({
    model: 'llama2-chinese',
    prompt: userCommand,
    stream: false
  })
})
```

#### 选项3: 混合模式（推荐）
```javascript
// 优先使用本地Ollama
// 如果本地不可用，回源到Claude API
// 既能离线工作，也能在需要时使用强大功能
```
