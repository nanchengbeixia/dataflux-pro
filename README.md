# 📦 DataFlux Pro v2.0 - 智能文件和表格处理系统

> **完整的Windows桌面应用，开箱即用！**
> 
> 一个集Excel实时处理、批量文件管理、AI助手、智能数据关联于一身的强大工具。

## 🎯 核心特性

### ✨ 主要功能

- **📊 Excel实时处理** - 与打开的Excel无缝协作，实时查看处理结果
- **📁 批量文件管理** - 智能重命名、分类、整理，省时又省力
- **🔗 AI智能助手** - 用中文说出需求，AI自动理解并执行
- **💾 数据智能关联** - 跨表关联、模糊匹配、自动填充，0个手动操作
- **🖱️ 浮窗快速访问** - 悬浮窗口，快捷菜单，随时就绪

### 🚀 优势

```
✓ 零学习成本 - 用中文说话，AI理解执行
✓ 开箱即用 - exe安装即可使用，无需配置
✓ 实时联动 - 不用关闭Excel就能看到处理结果
✓ 智能匹配 - 处理数据差异和拼写错误
✓ 安全可靠 - 所有操作可撤销，有完整备份
```

## 📥 快速开始 (3步)

### 1️⃣ 获取exe安装包

**方案A: 直接使用已编译的exe（最快）**
```
如果已有编译好的exe文件：
1. 双击 DataFlux-Pro-Setup-v2.0.0.exe
2. 按照向导完成安装
3. 完成！应用已准备好使用
```

**方案B: 从源代码编译（开发者）**
```bash
# 1. 进入项目目录
cd dataflux-pro

# 2. 安装依赖（第一次需要）
npm install

# 3. 一键打包生成exe（约10-15分钟）
node build-and-package.js

# 完成！exe已生成在 release/ 文件夹
```

### 2️⃣ 启动应用

```
安装完成后：
1. 桌面找到 DataFlux Pro 快捷方式，双击启动
2. 或在开始菜单中找到 DataFlux Pro
3. 看到浮窗和主窗口，应用已就绪！

快捷键: Ctrl + Alt + D (显示/隐藏浮窗)
```

### 3️⃣ 开始使用

```
浮窗出现了！现在你可以：

1. 点击 📁 处理文件夹里的文件
2. 点击 📊 处理打开的Excel表格  
3. 点击 💬 输入中文命令给AI助手
4. 点击 ⚙️ 打开设置面板

就这么简单！
```

## 🐛 常见问题

### 打包时出现错误
```
→ 运行 node environment-check.js 检查环境
→ 确保 Node.js 和 npm 已正确安装
→ 尝试 npm install 重新安装依赖
→ 查看 build-and-package.js 的日志输出
```

### Excel功能不工作
```
→ 检查是否安装了 Office 或 WPS
→ 确保 Excel 文件格式正确 (.xlsx or .xls)
→ 重启应用试试
→ 查看设置中的 Office 检测结果
```

### AI助手无响应
```
→ 检查网络连接（如使用云端模式）
→ 确保已配置 API 密钥（如使用Claude）
→ 尝试使用本地Ollama替代
→ 查看应用日志了解具体错误
```

### 浮窗消失了
```
→ 按 Ctrl + Alt + D 重新显示
→ 在任务栏找到应用，右键选择显示
```

更多问题？查看 **完整使用教程.md** 的"常见问题"部分。
