## 项目概述

**DataFlux Pro** 是一款企业级Windows桌面应用，用于批量处理文件和Excel表格。

### 核心功能
- ✅ **表格数据分析与修改** - 智能识别、转换和修改Excel数据
- ✅ **批量文件处理** - 智能重命名、分类、移动文件
- ✅ **跨表数据关联** - 模糊匹配、自动填充、数据同步
- ✅ **高速查询引擎** - 百万级数据秒速响应
- ✅ **格式保持** - 复制数据时保持原格式，无乱码

### 技术栈
```
Electron + Vue 3 + Node.js + Vite
+ ExcelJS + Fuse.js + SQLite
```

## 快速启动

```bash

npm run build

# 打包为可执行文件 (.exe)
npm run package:win

# 打包为便携版本（无需安装）
```

## 核心功能开发

### 1. Excel表格处理

**文件**: `src/main/ipc-handlers/excel.ts`

```typescript
// 使用示例
const result = await window.electronAPI.excel.readFile('path/to/file.xlsx');
const transformed = await window.electronAPI.excel.transformData(
  data,
  [{ type: 'filter', field: 'age', condition: 'greaterThan', value: 18 }]

```

**支持的操作**:
- 读写Excel文件 (.xlsx, .xls)
- 数据类型识别
- 条件筛选
- 数据转换和清洗
- 单元格格式操作

### 2. 文件批量处理

**文件**: `src/main/ipc-handlers/files.ts`

```typescript
// 使用示例
const result = await window.electronAPI.files.renameFiles([
  { oldPath: 'file1.txt', newPath: 'new_file1.txt' },
  { oldPath: 'file2.txt', newPath: 'new_file2.txt' }
]);
```

**支持的操作**:
- 批量重命名
- 批量移动/复制
- 智能分类
- 文件信息获取

### 3. 跨表数据关联

**文件**: `src/main/ipc-handlers/search.ts`

```typescript
// 使用示例 - 关键功能！
const result = await window.electronAPI.search.crossTableJoin({
  sourceTable: table1,
  targetTable: table2,
  sourceKey: 'name',
  targetKey: 'employee_name',
  matchFields: ['department'],
  threshold: 0.85
});

// 数据填充
const filled = await window.electronAPI.search.fillData({
  sourceData: table2,
  targetData: table1,
  mappings: [
    { sourceField: 'salary_account', targetField: 'bank_account' }
  ],
  joinKey: 'employee_id'
});
```

**核心算法**:
- **Levenshtein距离**: 计算字符串相似度（处理拼写差异）
- **Fuse.js模糊搜索**: 快速全文搜索
- **加权评分**: 多字段综合匹配
- **索引加速**: 百万级数据秒速查询

### 4. 系统与环境检测

**文件**: `src/main/ipc-handlers/system.ts`

```typescript
// 使用示例
const envCheck = await window.electronAPI.system.checkEnvironment();
const officeInfo = window.electronAPI.system.checkOfficeInstalled();
```

**检测内容**:
- Windows版本
- Node.js版本
- Office/WPS安装状态
- .NET Framework版本
- 系统库完整性

## 开发规范

### 1. IPC通信规范

所有Electron主进程与渲染进程的通信必须通过预加载脚本:

```typescript
// ✅ 正确方式
const result = await window.electronAPI.excel.readFile(filePath);

// ❌ 错误方式 - 不要直接使用ipcRenderer
// 这会违反安全性原则
```

### 2. 类型定义

为所有IPC调用定义类型:

```typescript
// src/shared/types/ipc.ts
export interface ExcelReadResult {
  success: boolean;
  content?: Record<string, SheetData>;
  error?: string;
}

export interface SheetData {
  headers: string[];
  data: Record<string, any>[];
}
```

### 3. 错误处理

```typescript
try {
  const result = await window.electronAPI.excel.readFile(filePath);
  if (!result.success) {
    showError(result.error);
    return;
  }
  // 处理成功情况
} catch (error) {
  showError('操作失败: ' + error.message);
}
```

### 4. 组件通信

使用Pinia状态管理:

```typescript
// store/index.ts
import { defineStore } from 'pinia';

export const useStore = defineStore('main', {
  state: () => ({
    currentFile: null,
    tables: [],
    selectedRows: []
  }),

  getters: {
    totalRows: (state) => state.tables.reduce((sum, t) => sum + t.data.length, 0)
  },

  actions: {
    async loadFile(filePath: string) {
      const result = await window.electronAPI.excel.readFile(filePath);
      if (result.success) {
        this.tables = Object.values(result.content);
      }
    }
  }
});
```

## 常见问题

### Q1: 启动时提示"Office未安装"

**解决方案**:
```bash
# 检查是否正确安装了Excel或WPS
# Windows + R，输入 excel 或 et 尝试打开应用

# 如果仍不能识别，手动检查路径
# Excel: C:\Program Files\Microsoft Office
# WPS: C:\Program Files\WPS Office
```

### Q2: 导入数据时出现乱码

**解决方案**:
```typescript
// SearchService已处理编码问题
// 自动处理UTF-8和GB2312转换
// 移除控制字符
const sanitized = sanitizeValue(cellValue);
```

### Q3: 大文件处理速度慢

**优化方案**:
```typescript
// 1. 使用索引加速查询
const indexes = await search.createIndex(data, ['name', 'id']);

// 2. 使用批量查询
const results = await search.batchQuery(indexes, queries);

// 3. 限制数据范围
const filtered = data.filter(row => row.date > '2024-01-01');
```

### Q4: 模糊匹配准确度不够

**调整方案**:
```typescript
// 提高阈值（更严格）
const result = await search.crossTableJoin({
  threshold: 0.95  // 从0.85改为0.95
});

// 或增加匹配字段
matchFields: ['department', 'position', 'hire_date']
```

### Q5: 打包后文件很大

**优化方案**:
```bash
# 使用便携版本（单文件）
npm run package:portable

# 或删除不必要的依赖
npm prune --production
```

## 进阶开发

### 添加新的IPC处理器

```typescript
// 1. 在 src/main/ipc-handlers/custom.ts 创建新处理器
export function registerCustomHandlers(ipcMain: any) {
  ipcMain.handle('custom:do-something', async (event, args) => {
    return { success: true, result: args };
  });
}

// 2. 在主进程注册
import { registerCustomHandlers } from './ipc-handlers/custom.js';
registerCustomHandlers(ipcMain);

// 3. 在预加载脚本添加API
// src/main/preload.ts
custom: {
  doSomething: (args: any) =>
    ipcRenderer.invoke('custom:do-something', args)
}

// 4. 在前端使用
const result = await window.electronAPI.custom.doSomething(data);
```
