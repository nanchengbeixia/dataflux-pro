# 📦 DataFlux Pro - 完整安装和使用教程

## 📥 第一部分：获取和安装

### 方案A：直接使用已编译的exe（推荐）

#### 1. 下载安装包
```
文件名: DataFlux-Pro-Setup-v2.0.exe (约80MB)
来源: 官网下载或网盘链接
```

#### 2. 安装过程
```
双击 DataFlux-Pro-Setup-v2.0.exe
   ↓
[欢迎界面] → 点击"下一步"
   ↓
[环境检测] → 自动检查Windows/Node.js/Office
            → 如缺失，自动下载安装
   ↓
[选择安装位置] → 推荐: C:\Program Files\DataFlux Pro
   ↓
[创建快捷方式] → 勾选 ✓ 创建桌面快捷方式
               → 勾选 ✓ 添加到开始菜单
   ↓
[安装] → 进度条显示中... 
   ↓
[完成] → ✓ 安装完成！
   ↓
[启动应用] → 自动打开DataFlux Pro
```

#### 3. 首次启动
```
启动应用（首次3-5秒，之后1-2秒）
   ↓
看到欢迎界面
   ↓
看到浮窗和主窗口
   ↓
已准备好使用！
```

### 方案B：从源代码编译（开发者）

```bash
# 1. 克隆项目
git clone https://github.com/yourname/dataflux-pro.git
cd dataflux-pro

# 2. 安装依赖
npm install

# 3. 开发模式运行
npm run dev

# 4. 生成exe（需要几分钟）
npm run package:win
```

## 💡 第三部分：实战操作案例

### 案例1：Excel表格求和

**需求**: 将A1和A2的值相加，结果放在A3

**操作步骤**:

```
1. 打开Excel表格
   ├─ 创建测试数据
   │  ├─ A1: 10
   │  ├─ A2: 20
   │  └─ A3: (空)
   └─ 保持Excel打开

2. 打开DataFlux Pro浮窗
   └─ 点击"D"展开浮窗

3. 点击"📊 Excel处理"
   └─ 主窗口打开，自动检测到你的Excel

4. 在AI输入框输入命令
   └─ 输入: "把A1和A2加起来放到A3"

5. 按Enter或点击发送
   └─ AI理解你的意图
   └─ 自动生成公式: =A1+A2

6. 点击"执行"
   └─ DataFlux Pro立即在你的Excel中：
      ├─ 在A3单元格写入公式
      ├─ 计算结果: 30
      └─ 自动保存

7. 回到Excel表格
   └─ 你会看到 A3 显示 30
   └─ 没有任何中断！
```

**效果演示**:
```
Excel表格 ────────────────────────────────────

A1: 10
A2: 20
A3: 30  ← 自动计算并填充！

DataFlux 浮窗 ────────────────────────────────

[执行完成！ 操作时间0.5秒]
```

## 🐛 第六部分：常见问题

### Q1: 浮窗消失了怎么办？
**A**: 按 Ctrl + Alt + D 重新显示
   或在任务栏找到 DataFlux Pro，右键选择"显示"

### Q2: Excel数据没有更新？
**A**: 
```
1. 确保Excel文件已保存
2. 检查Excel是否处于编辑模式
3. 重启DataFlux Pro
4. 或者手动点击"刷新"按钮
```

### Q3: AI命令没有反应？
**A**:
```
1. 检查网络连接（如使用云端模式）
2. 检查Ollama是否运行（本地模式）
3. 查看日志文件了解具体错误
4. 尝试更明确的命令描述
```

### Q4: 文件操作出错了怎么办？
**A**:
```
1. 点击浮窗中的"撤销"按钮
2. 或按 Ctrl + Z
3. DataFlux Pro会自动恢复到之前的状态
4. 所有操作都有备份（可在设置中查看）
```

### Q5: 性能很慢？
**A**:
```
1. 关闭不必要的后台应用
2. 检查CPU和内存占用
3. 减少监控的文件夹数量
4. 使用本地AI模式而不是云端
5. 清理临时文件
```

### Q6: 如何卸载？
**A**:
```
Windows 设置 → 应用 → 应用和功能
找到 DataFlux Pro → 卸载
所有操作历史和设置也会被删除
```

### Q7: 安装时提示"Office未安装"？
**A**:
```
1. 安装 Microsoft Office 或 WPS
2. 或者在设置中跳过此检查
3. 部分功能可能无法使用，但基础功能可用
```
